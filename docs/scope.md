# Scope

## P0: 반드시 해야 함

- 지도에서 카페 핀 표시
- 핵심 커피 필터 6개 이상
- 카페 상세의 가능한 커피, 신뢰도, 최근 확인일
- 로그인 사용자의 카페 저장/해제와 기본 저장 리스트
- 네이버지도/카카오맵/Google Maps 링크아웃
- 정보 수정 제보
- 관리자 페이지: 카페 CRUD, 태그 관리, 제보 승인/반려, CSV Import

## P1: 베타 이후

- 메뉴판 이미지 업로드
- OCR
- 사장님 인증
- 공유 리스트
- 서울 확장용 지역/운영 기준 정리
- 카페별 변경 이력
- CSV Export와 Evidence 이미지 관리 고도화

## P2: 하지 않음

- 리뷰/별점
- 주문/결제
- 전국 카페 DB
- SNS 피드
- 복잡한 추천 알고리즘

## 레트로 별관 전용 실험

방문 도장(방문 기록), 개인 메모, 코스 같은 개인 기록 기능은 레트로 별관(`/retro`)에서만 제공하는 실험 기능이며, 본 서비스 스코프(P0/P1)에 포함하지 않는다. 본 서비스 편입 여부는 클로즈드 베타 이후 재평가하며, 편입 전에는 저장 데이터 모델(`saved_lists`/`saved_cafes`)을 확장하지 않는다.
