# BrewMap Retro UI/UX QA Checklist

이 체크리스트는 브루맵 테마가 적용된 레트로 UI가 기존 MVP 동작을 해치지 않고 동일한 핵심 기능을 제공하는지 점검하기 위해 작성했습니다.

## 점검 항목

- [x] 레트로 데스크톱이 단일 앱 화면으로 동작하며, 스크롤을 내려도 이전 테스트/레거시 UI가 노출되지 않는다.
- [x] 기존 검색/결과/지도/저장/제보/Admin DOM은 JavaScript 상태와 접근성 보조용으로 유지하되 화면 흐름에서는 숨긴다.
- [x] 상단 레트로 메뉴와 태스크바가 레거시 섹션으로 스크롤하지 않고 대응 프로그램 창을 연다.
- [x] 카페 인덱스는 기존 MVP 필터와 저장 상태를 반영하고, 필터 초기화 및 카페 선택이 정상 동작한다.
- [x] 지도 프로그램은 데이터 기반 핀 선택, 선택 카페 상세, 저장, 외부 지도 링크를 제공한다.
- [x] 모바일에서는 창을 하나씩 전체 화면 카드처럼 표시해 화면 폭/배치 문제를 줄인다.
- [x] 창 위치는 뷰포트 변화에 맞춰 안전 영역 안으로 보정된다.
- [x] 저장/방문 기록 상태 변경 후 레트로 UI와 기존 상태 저장이 동기화된다.
