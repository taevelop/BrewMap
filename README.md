# BrewMap

BrewMap은 취향과 방문 맥락을 기준으로 스페셜티 커피 매장을 탐색하는 지도 기반 웹 애플리케이션 프로토타입입니다.

## 주요 구성

- **카페 지도 미리보기**: 현재 위치 기반으로 주변 카페 핀과 경로를 표현합니다.
- **취향 필터 검색**: 스페셜티, 핸드드립, 작업하기 좋은 카페 등 주요 조건을 빠르게 선택합니다.
- **추천 카페 카드**: 평점, 동네, 로스팅 성향, 방문 거리, 태그를 카드로 비교합니다.
- **방문 기록 확장 기반**: 원두 노트와 리뷰 기능을 붙일 수 있도록 기능 영역을 분리했습니다.

## 개발 명령어

```bash
npm install
npm run dev
npm run build
npm run lint
```
